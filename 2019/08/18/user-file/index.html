补充延伸

passwd和shadow文件比较

  • 二者都使用了(:)作为分隔符, 以回车作为一条记录的结束
  • 二者除了第一个域都是用户名外, 其它域的信息都不同
  • passwd 关注的是用户的细节信息, shadow关注的是密码的信息
  • passwd的信息每个用户都可以读取, shadow的信息只有root用户才能读取
  • 用户的加密密码只能存放于shadow中
  • passwd存放的信息近乎静态很少修改, shadow的信息更新得较为频繁
