基础篇章

工作原理

  1. 从文件从读取一行, 放入pattern-space –> Read
  2. 执行处理指令 –> Execute
  3. 打印pattern-space内容 –> Print
  4. 重复1-3步骤, 如读取到文件末尾, 终止退出 –> Repeat

基本语法

基本命令

打印pattern-space(p 命令)

  • p命令 用于打印出pattern-space的内容

  • 由于sed默认打印出pattern-space空间内容

  • 直接使用p命令, 将会得到二次打印的结果

  • -n选项能关闭sed默认打印的功能

  • 因此p命令常与-n选项结合使用

地址范围定义

行范围定义 (,)
自定义地址范围 (+, ~, //)

删除行(d 命令)

  • 所谓删除, 意味着不再输出中显示
  • 原文件内容并不会被删除
  • d命令与p命令类似, 均可以与地址匹配一起使用

将pattern-space内容写入文件 (w 命令)

  • 想对于将pattern-space中内容直接输出
  • w命令可以将其, 输出到指定文件中
  • (虽然许多人可能喜欢使用重定向解决 但sed的确提供了这个功能)
