前置知识

awk版本

  • awk是一门用作文字处理的特定语言, 常用于数据提取和报告生成

  • awk是最原始的awk实现版本

  • gawk是GNU awk的简称, 是awk的GPL版本

  • 由于awk为人所熟知, 因此linux下awk命令实际上是gawk的软链接

  • awk还有其他分支版本nawk, mawk

    • nawk: 对awk语言进行了更新
    • mawk: awk编程语言的解释器

基础名词

  • 默认情况下, \n换行符分割为一行
  • 一行内容, 在awk中被称为一条记录(record)
  • 默认情况下, \t和空格是一个字段的分隔
  • 一个分割, 在awk中被称为一个域(field)
  • 行和域的标号, 均从1开始

工作原理

  • 常规模式

    1. 从文件从读取一行 –> Read
    2. 查看是否与模式匹配 –> /pattern/
    3. 执行awk命令操作代码 –> Execute
    4. 重复1-3步骤, 如读取到文件末尾, 终止退出 –> Repeat

  • 代码块模式

    1. 执行BEGIN代码块命令
    2. 从文件从读取一行
    3. 执行awkbody代码块操作
    4. 重复2步骤, 如读取到文件末尾, 结束操作
    5. 执行END代码块命令

程序结构

  • awk代码可被分为三个模块

  • BEGIN

    • 结构格式: BEGIN { awk-command }
    • 介绍
      • 是可选内容
      • BEGIN必须大写
      • 可写入多条命令
      • 常用于打印报告头部信息, 初始化变量

  • body

    • 结构格式: /pattern/ {action}
    • 介绍
      • awk命令执行的主体

  • END

    • 结构格式: END { awk-command }
    • 介绍
      • 是可选内容
      • END必须大写
      • 可写入多条命令
      • 常用与打印脚注内容, 以及清空操作

记录分割符 RS/ORS

RS

  • RS (input Record Separator) 输入记录分割符
  • 默认情况下, 换行被作为RS
  • 定义RS

ORS

  • ORS (Output Record Separator) 输出记录分割符
  • 默认情况下, 换行被作为ORS
  • 定义ORS

编号 NF, NR/FNR

NF

  • NF (Number of Fiedl) 域编号
  • 计算出以分割符划分出的域数量
  • 常用$NF 打印出最后一个域

NR

  • NR (Number of Record) 记录编号
  • 在循环中十分有用, 我们可以在循环体中打印出行号
  • 在END代码块中使用时, 可以统计文件的行数

FNR

  • FNR (File Number of Record) 文件记录编号
  • 对于不同的文件, 重新计算文件记录编号

文件名 FIRENAME

  • 文件名称
  • 常用于输出打印
