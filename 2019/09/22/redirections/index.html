文件描述符(file-descriptor)

  • 定义解释
    • 文件描述符是一个数字, 用于唯一标识计算机操作系统中打开的文件
    • 它描述了数据资源, 以及如何访问该资源
  • 场景
    • 当程序需要打开一个文件或资源的时候
    • 操作系统内核授予访问权限, 在全局文件表中创建一个条目
    • 反馈给程序相应的条目地址
  • 碎片式理解
    • 文件描述符是文件或输入/输出资源的抽象标识, 被简写为fd
    • 存在于unix-like的操作系统之中
    • 也被应用于windows上, 被称为file handle句柄
    • 文件描述符用一个非负的整数表述, 如 0, 1, 2, 255
    • 每个系统进程, 都将默认分配0, 1, 2, 三个文件描述符
      • 具体分配如下
        fd Name Abbreviation Description Example
        0 Standard input stdin 默认标准输入 pipline(
        1 Standard output stdont 默认标准输出 print命令, 键盘显示
        2 Standard error stderr 默认标准错误输出 命令不存在提示, 屏幕显示
